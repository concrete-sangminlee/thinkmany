# 연구 많이 된다. 잘 때도 생각날 거야

> 박사 과정의 시행착오를 줄이기 위한 60개 챕터. 논문, 실험, AI, 멘탈, 커리어를 한국 공학 대학원 현실에 맞춰 엮은 실전 가이드.

**🌐 라이브 사이트**: https://concrete-sangminlee.github.io/thinkmany/

---

## 이 책은 무엇인가

박사 과정의 5-7년은 인생에서 가장 밀도 높고도 불확실한 시간이다. 무엇을 배워야 하는지, 무엇을 피해야 하는지, 어떻게 지도교수와 소통하고, 언제 논문을 내고, 어떻게 멘탈을 지키는지 — 이런 핵심은 대개 명시적으로 가르쳐지지 않는다. 이 저장소는 그 시행착오를 뒤늦게 각자 배우지 않도록, 박사 과정의 실제 의사결정 지도를 먼저 펼쳐 놓는 데 목적이 있다.

이 저장소가 다른 박사 가이드와 갈라지는 지점:

- **한국 공학 대학원 현실을 정면으로 다룬다**. 위계, 회식, 학연, 병역, BK21, 과제 중심 운영처럼 실제로 부딪히는 문제를 피하지 않는다.
- **추상적 조언보다 결정 프레임을 준다**. 단계, 체크리스트, 비교표, 대응 원칙으로 바로 판단할 수 있게 구성했다.
- **AI 시대의 박사 과정을 반영한다**. ChatGPT, Claude, Copilot을 어떻게 도구로 쓰고 어디서 선을 그을지까지 포함한다.
- **읽는 책이 아니라 탐색하는 웹북에 가깝다**. 검색, 북마크, 진행률, PDF 저장을 통해 필요한 챕터로 바로 이동할 수 있다.
- **박사 전후를 한 흐름으로 묶는다**. 입학 전 준비부터 박사 후 첫 1년, 그 이후의 학습 자세까지 이어서 다룬다.

ch01-ch04, ch47, ch49, ch57-ch58

### Part 2. 논문을 읽고 쓰는 법
연구의 핵심 활동. 논문 읽기의 기술, 연구 문제 찾기, 글쓰기, 그림 제작.

ch11-ch14, ch36, ch45

프로그래밍, 버전 관리, 시뮬레이션, 문서 도구, 통계, 정보 관리.
ch15-ch19, ch35, ch50, ch52-ch53

### Part 5. AI와 머신러닝 for 공학 연구
AI 입문부터 실전까지. 머신러닝·딥러닝·PINN·생성형 AI·시계열·AI 시대의 박사.
ch20-ch26, ch37, ch54

### Part 6. 학습 리소스
교과서, 강의, 필독 논문, 공개 데이터셋, 학파와 계보.
ch27-ch28, ch60

### Part 7. 연구 윤리와 협업
연구 윤리, 협업, 오픈 사이언스, 지식재산, 피어 리뷰, 과학 커뮤니케이션.
ch29-ch30, ch38, ch40, ch42, ch44

### Part 8. 성장과 졸업 이후
지도교수 관계, 멘탈, 신체 건강, 결혼·가족, 위기 대응, 커리어, 평생 학습.

## 사이트 특징

- **검색·북마크·진행률**: 챕터별 북마크 저장, 읽기 진행률 추적, 전체 텍스트 검색.
- **다크 모드**: 낮·밤 모두 읽기 편한 디자인.
- **반응형**: 모바일·태블릿·데스크탑 모두 지원.
- **오프라인 지원**: Service Worker로 재방문 시 빠른 로딩.
- **PDF 저장**: 브라우저 인쇄 기능으로 로컬 보관 가능.
- **키보드 단축키**: `j`/`k` 이동, `/` 검색, `t` 사이드바, `d` 테마, `?` 도움말.

## 기술 스택

- **정적 사이트** — 빌드 단계 없음
- **Vanilla JS** + `marked.js` + `DOMPurify` + `mermaid.js` (모두 CDN)
- **GitHub Pages** 자동 배포 (main 푸시 시)
- 모든 챕터는 `content/chNN-*.md` Markdown 파일

로컬 실행이 필요하면 `python3 -m http.server 8000` 정도로 충분.
