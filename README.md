# 연구 많이 된다. 잘 때 생각 많이 날거야

> 대학원생을 위한 연구 종합 가이드북 — ML/AI, 연구 방법론, 도구, 커리어까지

**🌐 라이브 사이트**: https://concrete-sangminlee.github.io/thinkmany/

## 이 책은 무엇인가

박사 과정의 5-7년은 인생에서 가장 집중적이면서 가장 불확실한 시기다. 무엇을 배워야 하는지, 무엇을 피해야 하는지, 어떻게 지도교수와 소통하고, 언제 논문을 내고, 어떻게 멘탈을 지키는지 — 이 모든 것이 명시적으로 가르쳐지지 않은 채 학생이 5-7년을 거쳐 스스로 깨닫는다. 이 책은 그 깨달음의 궤적을 미리 지도로 만든 것이다.

기존 박사 가이드와 다른 점:

- **본인(reader)에 초점**. "당신"이나 "학생"이 아닌 "본인"을 주어로 쓰며, 독자가 매 순간의 결정자임을 강조한다.
- **한국 공학 대학원 맥락에 특화**. 위계, 회식, 학연, 병역, BK21, 과제 중심 운영 등 한국 연구실의 고유한 상황을 정면으로 다룬다. 동시에 글로벌 학계 기준도 함께 제시한다.
- **실용적 결정 프레임**. 추상적 조언이 아니라 구체적 단계·체크리스트·의사결정 표로 구성.
- **AI 시대의 박사 과정 반영**. ChatGPT·Claude·GitHub Copilot이 박사 과정을 어떻게 바꾸는가, AI를 어떻게 도구로 쓰면서 본인의 학자성을 유지하는가.
- **평생의 관점**. 박사 과정만이 아니라 박사 입학 전부터 박사 후 30년까지의 궤적을 함께 다룬다.

ch15-ch19, ch35, ch50, ch52-ch53

교과서, 강의, 필독 논문, 공개 데이터셋, 학파와 계보.

### Part 7. 연구 윤리와 협업
연구 윤리, 협업, 오픈 사이언스, 지식재산, 피어 리뷰, 과학 커뮤니케이션.

## 사이트 특징

- **검색·북마크·진행률**: 챕터별 북마크 저장, 읽기 진행률 추적, 전체 텍스트 검색.
- **다크 모드**: 낮·밤 모두 읽기 편한 디자인.
- **반응형**: 모바일·태블릿·데스크탑 모두 지원.
- **오프라인 지원**: Service Worker로 재방문 시 빠른 로딩.
- **PDF 저장**: 브라우저 인쇄 기능으로 로컬 보관 가능.
- **키보드 단축키**: `j`/`k` 이동, `/` 검색, `t` 사이드바, `d` 테마, `?` 도움말.

## 기술 스택

- **정적 사이트** — 빌드 단계 없음
- **Vanilla JS** + `marked.js` + `DOMPurify` + `mermaid.js` (모두 CDN)
- **GitHub Pages** 자동 배포 (main 푸시 시)
- 모든 챕터는 `content/chNN-*.md` Markdown 파일

로컬 실행이 필요하면 `python3 -m http.server 8000` 정도로 충분.
